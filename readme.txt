TLM analyze / MRAM Command Generator 使い方

作成日時: 2026-05-27 22:45:41 +09:00

概要
このツールは、MRAM変更用のJSONコマンドを生成するGUIツールです。
起動後に表示される選択ウィンドウで、TLEから生成する画面、または一般的なMRAM変更用の画面を選択できます。

起動方法
1. Start_TLM_analizer.bat をダブルクリックします。
2. 初回起動時は .venv が自動作成され、requirements.txt の依存ライブラリがインストールされます。
3. 「起動モード選択」ウィンドウが表示されます。

選択ウィンドウの使い方
1. 「TLEからMRAM変更コマンド生成」にチェックを入れると、TLE用ウィンドウが表示されます。
2. 「一般的なMRAM変更コマンド生成」にチェックを入れると、通常のMRAM変更用ウィンドウが表示されます。
3. チェックを外すと、対応するウィンドウが閉じます。
4. 片方をチェックすると、もう片方のチェックは自動で外れます。

TLEからMRAM変更コマンドを生成する場合
1. 5桁のNORAD IDを入力します。
2. AM_INITIALIZE_APP app_id を入力します。
3. 「TLEからJSON生成」を押します。
4. CelesTrakからTLEを取得し、MRAM変更用JSONを生成します。

TLE生成時の出力ファイル
NORAD IDが 68798 の場合、以下の2ファイルが出力されます。
- TLE_cmd_68798_1U.json
- TLE_cmd_68798_2U.json

一般的なMRAM変更コマンドを生成する場合
1. 出力ファイル名を入力します。
2. ベースアドレスを10進数で入力します。
3. AM_INITIALIZE_APP app_id を入力します。
4. 書き込む16進数データを入力します。
   スペースと改行は無視されます。
5. 「カスタムJSON生成」を押します。

一般MRAM生成時の出力ファイル
出力ファイル名を CUSTOM_cmd.json にした場合、以下の2ファイルが出力されます。
- CUSTOM_cmd_1U.json
- CUSTOM_cmd_2U.json

追加でインストールするライブラリ
"requirements.txt"に使用する外部ライブラリの一覧を管理しています。
使いたいものがあればここに追加してください。

注意事項
- 作成したコマンド類はU/Lする前に必ずスタック等で動作確認すること
- TLEから生成する場合は、インターネット接続が必要です。
- 16進数データには 0-9, A-F の文字を入力してください。
- 書き込む値のバイト数が大きい場合は自動でコマンドを複数に分割します。
- 出力ファイルは、この readme.txt と同じフォルダに生成されます。
- 生成済みの同名ファイルがある場合は上書きされます。
